# Home Builder 5

!!! warning "Add-on in Development"
    This add-on is currently being migrated to work with Blender 5.0 

A professional Blender add-on for architectural kitchen, bath, and closet cabinet design.

## Features

### Room Management
- Create and manage multiple rooms as separate scenes
- Switch between rooms easily
- Duplicate rooms with all contents
- Preserves unit and snapping settings across rooms

### Room Layout Tools
- **Walls**: Draw interior, exterior, half, and fake walls with customizable height and thickness
- **Doors & Windows**: Place doors and windows on walls
- **Floor & Ceiling**: Generate floor and ceiling geometry
- **Lighting**: Add room lighting
- **Obstacles**: Define obstacles for cabinet placement (TODO)

### Product Library
- **Frameless Cabinets**: European-style frameless cabinet construction
- **Face Frame Cabinets**: Traditional face frame cabinet construction (TODO)
- **Closets**: Closet system components (TODO)

### Layout Views
Create professional 2D documentation:
- **Elevations**: Generate wall elevation views
- **Floor Plans**: Top-down plan views
- **3D Views**: Perspective and isometric views
- **Multi-View Layouts**: Cabinet group layouts with plan and elevation views

#### Page Settings
- Multiple paper sizes (Letter, Legal, Tabloid, A4, A3)
- Landscape/Portrait orientation
- Adjustable scale (1/4", 3/8", 1/2", 3/4", 1", 1-1/2", 3")
- Fit to content
- Render to image

### Annotations
- **Dimensions**: Add linear dimensions with vertex snapping
  - Works in layout views (plan and elevation)
  - Works in 3D view from any angle (auto-detects XY, XZ, or YZ plane)
  - Three-click workflow: first point, second point, leader placement
  - Live preview with dimension text
  - Fractional inch display

## Installation

1. Download the add-on
2. In Blender, go to Edit > Preferences > Add-ons
3. Click "Install" and select the downloaded file
4. Enable "Home Builder 5" in the add-ons list

## Usage

All tools are accessible from the **Home Builder** tab in the 3D View sidebar (press N to toggle).

### Panel Organization
1. **Rooms**: Manage room scenes
2. **Room Layout**: Draw walls, place doors/windows, add floors
3. **Product Library**: Browse and place cabinets
4. **Layout Views**: Create and manage 2D documentation
5. **Annotations**: Add dimensions and notes
6. **Settings**: Configure add-on settings

## Requirements

- Blender 4.0 or later (developed with Blender 5.0)
